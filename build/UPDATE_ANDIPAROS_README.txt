Currently, there are no housekeeping functions for updating the configuration files.
In order to run this updated version of Andiparos properly you must delete the
following directory...

Windows 7 Users:	"C:/Users/<your_user_name>/Andiparos"
Windows XP Users:	"C:/Documents and Settings/<your_user_name>/Andiparos"
Linux Users:		"~/Andiparos"
